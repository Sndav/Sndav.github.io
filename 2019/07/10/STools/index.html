STools

什么是STools

STools是一套开源的，面向CTF竞赛的工具。

为什么要有STools

在CTF比赛中，我们常常会有一些重复性的劳动，或者需要协作的地方。为了方便比赛选手的进度共享，将当前进度保存，方便队友继续工作，笔者将开发一套工具。(十有八九会鸽，不要抱着太大的期待

基本的架构

架构这个东西，我也没啥想法。姑且分为FrontEnd,Middleware,Backend。

FrontEnd

前端主要是用户的交互嘛，就什么UI啊，控制啊，扫描啊。这些交互的东西。暂时应该会有Cli以及Web版本。(web版本十有八九会鸽,主要是前端太菜，如果有大佬想帮忙也行啊。

MiddleWare

这个东西就多啦，线下赛，抓包什么的

  1. 什么代理啊，线下赛不在现场的队友怎么办，就要用这个啊
  2. 抓包啊，Burp好用啊，但是不太好协同啊。
  3. 什么共享文档啊。
  4. 什么一键日卫星啊

BackEnd

这个东西啊，什么扫描器啊，什么的。主要是任务调度，还有什么插件啊，什么的。
