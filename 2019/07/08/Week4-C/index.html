题目

NCTF 2018 easy_audit

题解

访问网站

第一次绕过

这里过滤了几个问题一个$_REQUEST不能有字母，一个$_SERVER不能有关键字。

这是开始的两个绕过

针对第一个问题，不妨考虑变量覆盖，我们知道$_REQUEST可以接受COOKIE，POST的消息，我们不妨使用POST覆盖GET消息。

我们了解到$_SERVER不会对字符串解析，比如%27不会解析成‘我们可以利用这点绕过。

第二次绕过

针对md5我们不妨传入数组，便可以绕过

针对nctf的请求我们可以构造anctfisfun便可以绕过。

下面就是flag

因为我们使用了POST进行变量覆盖，我们便不能使用php://input进行绕过

我们有两种方法解决

  • 将POST覆盖改为COOKIE覆盖（我没实验成功）

  • 使用http协议从外部获取

因此我们可以在服务器上新建个网页

<?php echo "ccc_liubi";?>

为什么要这么写呢？

主要是如果直接是文本会产生一个换行符。然后匹配不上
