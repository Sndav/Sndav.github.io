    parse_str

    此函数不仅在php的自带函数之中，还用在$_POST,$_GET等变量的解析

    函数处理机制

    下面我将通过几个具体的例子来展开说明parse_str的一些具体的机制

    参数名处理

    PHP在接受参数名时，需要将怪异的字符串转换为一个有效的变量名，因此当进行解析时，它会做两件事：

    • 删除空白符
    • 将某些字符转换为下划线（包括空格）
    <?php

foreach(
    [
        "{chr}foo_bar",
        "foo{chr}bar",
        "foo_bar{chr}"
    ] as $k => $arg) {

        for($i=0;$i<=255;$i++) {
            echo "\033[999D\033[K\r";
            echo "[".$arg."] check ".bin2hex(chr($i))."";
            parse_str(str_replace("{chr}",chr($i),$arg)."=bla",$o);

            if(isset($o["foo_bar"])) {
                echo "\033[999D\033[K\r";
                echo $arg." -> ".bin2hex(chr($i))." (".chr($i).")\n";
            }
        }

        echo "\033[999D\033[K\r";
        echo "\n";
}

    得到

    {chr}foo_bar -> 20 ( )
{chr}foo_bar -> 26 (&)
{chr}foo_bar -> 2b (+)

foo{chr}bar -> 20 ( )
foo{chr}bar -> 2b (+)
foo{chr}bar -> 2e (.)
foo{chr}bar -> 5b ([)
foo{chr}bar -> 5f (_)

foo_bar{chr} -> 00 ()
foo_bar{chr} -> 26 (&)
foo_bar{chr} -> 3d (=)

    通过此方法我们就可以得到一些相应的绕过方法。
